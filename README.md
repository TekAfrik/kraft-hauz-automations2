## Local development

```bash
npm install
npm run dev
```

Then open `http://localhost:3000`.

## Scripts

- **dev**: start the local dev server
- **build**: create a production build
- **start**: run the production server locally
- **lint**: run ESLint checks

## Tech stack

- Next.js (App Router) + TypeScript
- Tailwind CSS
- Framer Motion

## Deploy

This repo is ready to deploy on Vercel. Set the project root to the repository root and keep the default Next.js settings.
